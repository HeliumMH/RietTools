**A collection of python scripts for Rietveld refinement using TOPAS**
- occ2uptake.py            :  Convert crystallgraphic occupancy to adsorption uptake: 
- topas_macro_Out_CIF.txt  : topas Out_CIF macros, copy+paste to local.inc, combine this with my other repo [TOPAS_v5_cif_cleaner](https://github.com/HeliumMH/TOPAS_v5_cif_cleaner) to clean and sort for checkcif
- export_table_from_cif.py : export entries in cif for reporting
